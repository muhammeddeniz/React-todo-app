# React Todo App


### **İçerdiği Paketler**

- [Mobx](https://mobx.js.org/) - _State Management_


### **Kurulum**
```sh
$ git clone https://github.com/muhammeddeniz/React-todo-app.git
$ cd React-todo-app
$ yarn
```

### **Geliştirme**

```sh
$ yarn start
```
**http://localhost:3000 adresinden projeyi geliştirmeye başlayabilirsiniz.**
